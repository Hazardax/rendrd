VULKAN

An open, cross-platform API for high-performance 3D graphics and computing, designed to provide developers with low-level control over modern GPU hardware and reduce CPU overhead compared to older APIs.

OPENGL

A cross-platform, hardware-independent API that allows programmers to render high-performance 2D and 3D vector graphics, serving as a standardized interface to graphics hardware across various platforms.

DIRECTX

A Microsoft-developed suite of APIs that serves as an intermediary between game software and hardware, standardizing commands for graphics, sound, and input to enable universal game development.
